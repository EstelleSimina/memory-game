Memory Game

Règles du jeu :

  1. Toutes les cartes sont placées face cachée sur le plateau de jeu.
  2. À chaque tour, vous pouvez retourner deux cartes.
  3. Si les deux cartes retournées sont identiques, elles restent face visible.
  4. Si les deux cartes retournées sont différentes, elles sont à nouveau retournées face cachée après un court délai.
  5. Le jeu continue jusqu'à ce que toutes les paires aient été trouvées.
